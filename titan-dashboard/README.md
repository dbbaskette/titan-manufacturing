# Titan Dashboard

React-based demo UI for Titan 5.0 AI Platform.

## Features
- Interactive architecture diagram with agent visualization
- Demo scenario triggers (Phoenix Incident, Boeing Expedite, FAA Audit)
- Real-time agent activity monitoring
- OpenMetadata lineage explorer
- Chat interface for natural language queries

## Screens
1. **Architecture View** — Interactive system diagram
2. **Facility Monitor** — 12 plant status overview
3. **Demo Scenarios** — Pre-built workflow triggers
4. **Chat Interface** — Talk to Titan 5.0
5. **Lineage Explorer** — OpenMetadata integration
6. **Agent Activity** — Real-time tool call monitoring

## Technology
- React 18
- Tailwind CSS
- Recharts for visualizations

## Environment Variables
```
REACT_APP_ORCHESTRATOR_URL=http://localhost:8080
REACT_APP_OPENMETADATA_URL=http://localhost:8585
```
